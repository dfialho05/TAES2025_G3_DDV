# WebSocket Game Server

Servidor WebSocket para jogo de cartas contra bot.

## Como usar

1. Instalar dependências:
```bash
npm install
```

2. Iniciar servidor:
```bash
npm start
```

3. Abrir `game.html` no navegador

4. Conectar e jogar!

## Arquivos principais

- `index.js` - Servidor principal
- `game.html` - Interface do jogo
- `core/` - Classes do jogo
- `handlers/` - Lógica dos websockets

O servidor roda na porta 3000.
